# 类的构造与方法调用

### 1. 类的构造

```lua
local a = clazz()
local b = claxx(1, "aaa")
```

- 直接通过 `类()` 进行实例化
- 构造函数参数可直接传入

---

### 2. 方法调用

#### 调用静态方法

```lua
clazz.func()
```

- 使用 `类.方法()` 调用静态函数

---

#### 调用非静态方法

```lua
clazz().func()
```

- 使用 `实例.方法()` 调用非静态函数

---

#### 调用私有方法

```lua
invoke(类或实例, "方法名", 参数...)
```

- 使用 `invoke` 调用`私有方法`或无法直接访问的方法
- 支持静态方法和实例方法，参数直接追加在后面
