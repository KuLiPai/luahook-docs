# Installation & Activation

## 1. Download LuaHook

Go to the GitHub [Latest Releases Page](https://github.com/KuLiPai/LuaHook/releases/latest), download the latest LuaHook installation package, and install it on your device.

---

## 2. Activation Methods

LuaHook currently supports two activation methods: **Root Activation** and **Shizuku Activation**.

---

### Method 1: Root Activation

1. Ensure your device has Root access and authorize LuaHook via a Root manager such as **KernelSU**.
2. Grant Root permission to LuaHook in your Root manager:
    ![Root Authorization](/assets/914a277b9eb506345a9bb0c1b40657c1-20250601145226-uzmjxa9.jpg)
3. After authorization, reopen LuaHook. If the interface looks like this, activation is successful:
    ![Root Activation Success](/assets/ccd3bcdb7cd254041f2c589e1da708f8-20250601144844-5a1tywe.jpg)

---

### Method 2: Shizuku Activation

1. Download and install [Shizuku](https://shizuku.rikka.app/), and follow the official guide to activate the Shizuku service.
2. Launch the LuaHook app. Upon first launch, an authorization prompt will appear:
    ![Shizuku Authorization Prompt](/assets/9b5350777372b7a5d06fa932bc22a1c7-20250601145118-yaqi83c.jpg)
3. Click **Always assume**, and after authorization, activation is successful, as shown below:
    ![Shizuku Activation Success](/assets/21fb5b406e901ecdef693e945173627f-20250601145156-di3euf3.jpg)
